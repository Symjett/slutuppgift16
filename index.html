Introduktion

Att veta hur man använder sig utav responsive images är ett bra sätt för att få användaren att kunna se en bild tydligare.
I kursen gränssnittsdesign använder man sig mycket utav responsiva sidor och vi tänkte då att kunskapen om responsive images
skulle vara användbart om man jobbar med webbutveckling i framtiden. Navigera i menyn ovan för att få reda på vad responsive images är!
